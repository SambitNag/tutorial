System Software:

System software refers to the programs and files that make up the operating system (OS) and its utilities. It manages hardware components, provides an environment for application software to run, and performs essential tasks to maintain the computer's operation.

Examples of system software include:

  1. Operating Systems : Windows, macOS, Linux, Android, iOS.
  2. Device Drivers : Software that allows the OS to communicate with hardware devices like printers, graphics cards, and network adapters.
  3. Utility Programs : Tools that help manage, maintain, and control computer resources. Examples include antivirus software, disk cleanup tools, and backup utilities.
  4. Firmware : Low-level software embedded in hardware devices, providing control, monitoring, and data manipulation of engineering systems.
